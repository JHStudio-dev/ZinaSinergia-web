# Zina Sinergia

Sitio web oficial de Zina Sinergia — marca personal de coaching, conferencias y procesos de transformación.

**Conecta · Inspira · Transforma**

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS

## Desarrollo

```bash
npm install
npm run dev
```

El sitio queda disponible en [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — entorno de desarrollo
- `npm run build` — build de producción
- `npm run start` — sirve el build
- `npm run lint` — análisis de código

## Estructura

```
src/
  app/         layout, estilos globales y página principal
  components/  layout, secciones y piezas de UI reutilizables
  data/        contenido editable del sitio
```
